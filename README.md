## 📌 Overview

**FitZone** is not just a gym website — it's a **digital fitness companion**. Built for both gym visitors and active members, this platform combines an informational front-facing website with a powerful member dashboard packed with AI-driven fitness tools. Whether you're looking to join a program, track your BMI, or get a personalized workout plan — GymPro has it all.

## ✨ Features

### 🔓 Public Pages (No Login Required)

| Page | Description |
|------|-------------|
| 🏠 **Home** | Landing page with gym highlights, quick shortcuts to Register, Plans, and Contact pages |
| ℹ️ **About** | Detailed information about the gym — its story, mission, and values |
| 💳 **Plans** | All available membership tiers with pricing and benefits clearly laid out |
| 🏃 **Programs** | Browse and directly register for specialized training programs |
| 🖼️ **Gallery** | Photo showcase of the gym's facilities, machines, and training areas |
| 📍 **Contact** | Branch-specific contact info, location details, and a Google Maps redirect |
| 📝 **Register** | Sign up for a membership plan with full personal details form |
| 🔐 **Login** | Secure login portal for registered members |

### 🔒 Member Dashboard (Login Required)

| Tool | Description |
|------|-------------|
| 📊 **BMI Calculator** | Instantly calculate your Body Mass Index |
| 🔥 **Calorie Calculator** | Find your daily calorie needs based on your fitness goal |
| 🤖 **AI Diet Planner** | Get smart, personalized diet plans powered by AI |
| 💪 **Workout Recommendation** | AI-generated workout plans tailored to your goals and fitness level |
| 🪪 **Membership Info** | View your current plan, membership status, and enrollment details |

## 💡 Why This Project Stands Out

Most gym websites are just static brochures. **GymPro is different.**

- 🤖 **AI-Powered Tools** — Real workout and diet recommendations, not just static content
- 🔐 **Full Auth System** — Secure registration and login with protected member routes
- 📍 **Google Maps Integration** — One click to navigate to the gym location
- 🏃 **Direct Program Registration** — Users can enroll in programs without leaving the site
- 📊 **Personal Fitness Dashboard** — Each member gets their own space with health tools and membership info

This project demonstrates real-world skills in **full-stack development, user authentication, API integration, AI tooling, and responsive UI design** — all shipped as one cohesive product.
